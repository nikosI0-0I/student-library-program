# 📚 RTU Library Full-Stack Web Application

An advanced C++ library management system featuring a real-time HTTP server engine and a clean web dashboard.

## 🛠️ Dependencies & Prerequisites
To compile and run this backend server, you must download and include the following header-only libraries in your local directory:
* **Crow C++ Framework (`crow_all.h`)**: Used for micro-routing and API generation.
* **Asio (`asio.hpp`)**: Used as the foundational asynchronous network I/O engine.
